Carbon Playground is an interactive learning tool to explore greenhouse gas effects and policy choices. Based on SDG 13 (Climate Action) and SDG 12.

Informative Interactive Minimalistic

What you’ll learn

Greenhouse gases add radiative forcing (W/m²).
Cutting methane reduces near-term warming.
Energy, waste, and forest actions support SDGs.

Model = simplified educational tool, not a forecast.

About the Website

This tool helps users visualize and spread awareness on how small changes made by each individual can make drastic changes in the climate of the world
